🛡️ AlertSakha (अलर्ट सखा)

**When Every Second Matters**

- Live Prototype Link: https://alertsakha.lovable.app

---

## 🚀 Project Overview

AlertSakha is a real-time emergency response platform designed for hospitality environments such as hotels and resorts. It enables fast and coordinated action between guests, staff, and administrators through a unified digital system.

The prototype is built using Lovable with integrated cloud backend and AI capabilities powered by Google Gemini.


---

##  Problem Statement

- Communication Silos: Critical information is often siloed, fracturing communication between stakeholders.
- Delayed Response: Manual reporting processes increase response time during high-stakes emergencies.
- Decentralized Coordination: Lack of a synchronized bridge between individuals and personnel.
- Detection Gaps: Difficulty in detecting emergencies quickly across decentralized ecosystems.

---

##  Solution

A role-based platform that provides:

- Instant SOS reporting
- Real-time coordination
- Centralized dashboards
- AI-assisted incident understanding

---

##  Key Features

- One-Tap SOS Button: Instant reporting for guests and users to trigger immediate alerts.
- Live Location Tracking: Real-time tracking to pinpoint exact incident coordinates.
- Real-Time Dashboards: Centralized monitoring for staff and admins to manage active emergencies.
- Instant Stakeholder Alerts: Automated notifications across the network via sub-second signaling.
- Evidence Collection: Capability to upload photo or video evidence directly to the emergency timeline.

---

##  User Roles

### 👑 Admin/ Hotel Manager (Restricted Access)

- Monitor all active and resolved emergencies via a central dashboard.
- Manage staff users and monitor notification logs.
- Coordinate with integrated emergency services (Police, Fire, Ambulance).

###  🛡️ Security / Hotel Staff

- Receive instant SOS notifications and incident assignments.
- Track live incident locations and provide real-time status updates.
- Access incident evidence for coordinated resolution.

###  🏨 Guest / User

- Use the one-tap SOS button to report incidents instantly.
- Provide incident descriptions and evidence.
- Receive status updates during an active crisis.

## 🛠️ Technology Stack

- Frontend: React.js, Tailwind CSS
- Backend: Lovable Cloud (managed backend)
- Database: Cloud database (auto-generated schemas)
- Authentication: Built-in email authentication
- Storage: Cloud storage for evidence uploads
- AI Integration: Google Gemini via Edge Functions
- Deployment: Lovable Publish (serverless hosting)
- Version Control: Git & GitHub

## 🎯 Future Enhancements

- Voice-Based Reporting: Hands-free emergency reporting for increased accessibility.
